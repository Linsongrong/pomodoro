# 🍅 Pomodoro Timer

一个简洁好用的番茄钟 Web 应用，单文件实现，零依赖，开箱即用。

## 功能

- **三种计时模式**：专注 25 分钟 / 短休息 5 分钟 / 长休息 15 分钟
- **圆形进度环**：直观显示倒计时进度
- **自定义时长**：可在设置面板修改三种模式的时长，刷新页面不丢失
- **任务记录**：开始前填写任务名，完成后自动记录到历史
- **今日统计**：显示完成番茄数和总专注时长
- **浏览器通知**：计时结束时弹出系统通知（需授权）
- **提示音**：用 Web Audio API 生成，无需音频文件
- **Tab 标题同步**：倒计时实时显示在浏览器标签页标题
- **响应式设计**：手机和桌面都能用

## 快速开始

```bash
git clone git@github.com:Linsongrong/pomodoro.git
cd pomodoro
python3 -m http.server 8765
```

打开浏览器访问 http://localhost:8765

## 使用说明

1. 在输入框填写当前要做的任务（可选）
2. 点击 **开始** 进入专注模式
3. 25 分钟结束后自动切换到短休息，每完成 4 个番茄切换长休息
4. 点击右上角 ⚙️ 可自定义时长
5. 点击「开启浏览器提醒」授权通知权限，计时结束时会弹出系统通知

## 技术栈

纯 HTML + CSS + JavaScript，单文件，零依赖，数据存储在 localStorage。
